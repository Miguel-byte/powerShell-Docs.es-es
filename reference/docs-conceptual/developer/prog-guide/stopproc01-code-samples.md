# <a name="stopproc01-code-samples"></a>Ejemplos de código StopProc01

Este es el ejemplo de código para el cmdlet de ejemplo StopProc01. Este es el ejemplo de cmdlet `Stop-Process` que se describe en [creación de un cmdlet que modifica el sistema](../cmdlet/creating-a-cmdlet-that-modifies-the-system.md). El cmdlet `Stop-Process` está diseñado para detener los procesos que se recuperan con el cmdlet Get-proc (descrito en [creación del primer cmdlet](../cmdlet/creating-a-cmdlet-without-parameters.md)).

> [!NOTE]
> Puede descargar el C# archivo de código fuente (stopproc01.CS) para el cmdlet Stop-proc mediante el kit de desarrollo de software de Microsoft Windows para Windows Vista y los componentes de tiempo de ejecución de .NET Framework 3,0. Para obtener instrucciones de descarga, consulte [Cómo instalar Windows PowerShell y descargar el SDK de Windows PowerShell](/powershell/developer/installing-the-windows-powershell-sdk).
>
> Los archivos de origen descargados están disponibles en el directorio **@no__t ejemplos de 1PowerShell >** .
