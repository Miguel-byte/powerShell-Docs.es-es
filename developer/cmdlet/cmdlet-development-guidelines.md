# <a name="cmdlet-development-guidelines"></a>Directrices de desarrollo del cmdlet

Los temas de esta sección proporcionan las instrucciones de desarrollo que puede usar para generar los cmdlets de formato correcto. Al aprovechar la funcionalidad común proporcionada por el tiempo de ejecución de Windows PowerShell y siguiendo estas instrucciones, puede desarrollar cmdlets sólidos con el mínimo esfuerzo y proporcionar al usuario una experiencia coherente. Además, se reducirá la carga de prueba porque la funcionalidad común no requiere volver a probar.
