# <a name="propertyname-element-for-selectioncondition-for-groupby-format"></a>Elemento PropertyName para SelectionCondition for GroupBy (formato)

Especifica la propiedad de .NET que desencadena la condición. Cuando esta propiedad está presente o cuando se evalúa como `true`, se cumple la condición y se usa la definición. Este elemento se usa al definir cómo se muestra un nuevo grupo de objetos.
