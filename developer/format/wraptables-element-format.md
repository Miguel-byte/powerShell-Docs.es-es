# <a name="wraptables-element-format"></a>Elemento WrapTables (formato)

Especifica que los datos de una celda de tabla se mueven a la línea siguiente si los datos están mayor que el ancho de la columna.

Configuración (formato) elemento DefaultSettings (formato) WrapTables elemento (formato)

## <a name="syntax"></a>Sintaxis

```xml
<WrapTables/>
```

## <a name="attributes-and-elements"></a>Atributos y elementos

Las secciones siguientes describen los atributos, elementos secundarios y el elemento primario de la `WrapTables` elemento.

### <a name="attributes"></a>Atributos

Ninguna.

### <a name="child-elements"></a>Elementos secundarios

Ninguna.

### <a name="parent-elements"></a>Elementos primarios

|Elemento|Descripción|
|-------------|-----------------|
|[Elemento DefaultSettings (formato)](./defaultsettings-element-format.md)|Define la configuración común que se aplica a todas las vistas del archivo de formato.|
